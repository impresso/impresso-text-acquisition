SWISSINFO PDF-embedded importer
===============================

This importer is a special case of PDF-embedded OCR which has been extracted into a custom JSON format. 
It was developed to handle OCR radio-bulletin data in PDF format provided by Memoriav from the `Swissinfo
<https://www.swissinfo.ch/eng/>`_ collection of World-War II radio-bulletins.
